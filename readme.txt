transaction_id - Unique identifier for each transaction

amount - Transaction amount

oldbalanceOrg - Sender's balance before the transaction

newbalanceOrig - Sender's balance after the transaction

oldbalanceDest - Receiver's balance before the transaction

newbalanceDest - Receiver's balance after the transaction

type - Transaction type (e.g., CASH_OUT, TRANSFER, etc.)

isFraud - Label indicating whether the transaction is fraudulent (0 or 1)

isFlaggedFraud - Flag indicating if the transaction was flagged as suspicious
